Using MySQL with Node.js: A Beginner's Guide

Step 1: Install Node.js and MySQL

Before we start, make sure you have Node.js and MySQL installed on your system.

Step 2: Create a New Node.js Project

Create a new Node.js project by running the following command in your terminal:

npm init -y

Step 3: Install MySQL Node.js Package

Install the MySQL Node.js package by running the following command in your project directory:

npm install mysql

Step 4: Setting Up MySQL Connection

Setup a connection to MySQL database by creating a file named "db.js" and adding the following code:

const mysql = require('mysql'); const connection = mysql.createConnection({ host: 'localhost', user: 'root', password: 'yourpassword', database: 'yourdatabase' }); connection.connect((error) => { if (error) { console.error('Error:', error); } });

Step 5: Running Queries

You can now run MySQL queries in your Node.js application. Create a file named "index.js" and add the following code to query data from your MySQL database:

connection.query('SELECT * FROM yourtable', (err, results) => { if(err) throw err; console.log(results); }); connection.end();

Step 6: Run Your Node.js Application

Run your Node.js application by using the following command in your terminal:

node index.js

This will display the results of your query in the console.
